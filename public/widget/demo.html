How to Embed on Your Website

Option 1: Simple HTML (Recommended)

<!-- Add this where you want the viewer -->
<div data-claypixels-viewer="YOUR_VIEWER_ID" style="width:100%;height:500px;"></div>

<!-- Add this before </body> -->
<script src="https://claypixels.eu/widget/widget.js"></script>

Option 2: JavaScript API

<div id="my-viewer"></div>

<script src="https://claypixels.eu/widget/widget.js"></script>
<script>
  ClayPixels.init({
    container: '#my-viewer',
    viewerId: 'YOUR_VIEWER_ID',
    onTextureUpload: function(data) {
      console.log('Texture uploaded!', data);
    }
  });
</script>

Enable Widget for Your Viewer

To use the widget, you need to enable it in your viewer settings:

  1. Go to your Viewer Admin Panel
  2. Select your viewer
  3. Open Settings
  4. Enable "Widget Embedding"
  5. Choose Storage Mode:
    • Hybrid (recommended) - Processed textures stored locally, originals backed up to cloud
    • Local - Everything stays on user's device
    • Server - Traditional server-side storage
  6. Copy your Viewer ID or Short Code
